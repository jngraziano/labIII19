map: le paso una funcion por cada uno de los elementos. Devuelve un array con la misma cantidad de elementos, pero modificamos el valor.
var n = nums.map(
function (num){

filter: sirve para tener una nueva lista filtrada por la condicion que le ponga. devuelve un array con menos cantidad de elementos.
var n = nums.filter(funcion(num){
	return num>3;
}

reduce: le pasa dos parametros una funcion y un valor 

var n = nums.reduce(function(total,num){
	return total+=num;
}
